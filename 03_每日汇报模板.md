# 每日学习汇报模板

每天结束后复制以下内容发给 Codex。没完成也如实填写，不为了打卡补写结果。

```text
日期：
计划学习时间：
实际学习时间：

今天完成：
1.
2.
3.

实际产出：
- 代码：
- UE工程/材质：
- 截图或视频：
- 笔记：

能否不看教程重做：
- 能：
- 部分能：
- 不能：

今天能用自己的话解释的内容：

遇到的问题或报错：

计划完成但没有完成：

证据路径或附件：
```

## Codex 固定反馈内容

每次汇报后应给出：

1. 今日判定：完成 / 部分完成 / 未完成。
2. 掌握情况：已经掌握、需要复习、存在误解。
3. 代码、材质节点或操作中的具体问题。
4. 未完成任务如何顺延，哪些加分项需要删除。
5. 下一学习日的课程范围、实践任务和验收问题。

## 判定标准

- 完成：有独立产出，并能解释关键原理。
- 部分完成：跟随教程完成，但不能独立重做，或缺少证据。
- 未完成：只有观看记录，没有代码、材质、工具或实际操作。
