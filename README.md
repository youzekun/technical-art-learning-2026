## 固定执行规则

1. 暑假每周学习6天、每天约5小时；开学后按每周约20小时执行；周日可以休息或轻量复盘，不补双倍课。
2. 看完视频不算完成，必须留下代码、UE 工程、材质、截图、视频、性能数据或文档。
3. 单日未完成时，下一学习日先处理未完成的必做项，并删除加分项。
4. 连续两天未完成时，减少教程量，保留独立实践与作品里程碑。
5. 12 月前不主攻 ZBrush、高级 Houdini、PBRT、BDPT、波动光学、DX12 或 UE 源码修改。
6. 先使用 UE 传统材质系统，不同时学习 Substrate。
