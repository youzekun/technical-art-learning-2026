- [x] 不看教程完成7题口头自测，能够解释`a`、`&a`、`p`、`*p`、`nullptr`、`const`指针和野指针。

```cpp
#include <iostream>

int main()
{
    int a = 10;
    int* p = &a;

    std::cout << "a = " << a << '\n';
    std::cout << "&a = " << &a << '\n';
    std::cout << "p = " << p << '\n';
    std::cout << "*p = " << *p << '\n';

    *p = 20;
    std::cout << "a after *p = 20: " << a << '\n';
    return 0;
}
```
